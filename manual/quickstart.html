Quickstart

Setup

Download (either the zip, tarball, or git cloned) Ding. Copy src/Ding directory to a path inside the include_path for your php installation.

Please see docs/examples/basic directory for the most basic setup.

Also, please see docs/examples/yaml directory for the most basic setup using yaml.

If you'd like to use Ding as a PHAR library, take a look at docs/examples/basic-with-phar
